# Home SOC Setup

---

## Summary:

I want to create a fully functional SOC on my home network. The hope of this project is to experiment
working with the ELK stack, and to test out some automations and more simple scripting.

So Far, I have...

- Bought Optiplex for Home Computer
- Made sure I could remote in with TeamViewer
- Created a VM with 18 GB of ram, 250 GB storage, 4 CPU cores to run the SOC off of
- I got the VM to run successfully with a Ubuntu OS
- I have downloaded the ELK stack onto the VM

Next Steps...

- Go through the setup process for Configuring the ELK tech stack
- Continue experimenting and testing out scripts
