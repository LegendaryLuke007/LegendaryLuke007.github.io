# NAS BACKUP SOLUTION:

## Summary:

This was a project brought to me by my father who owns an Engineering Business. He wanted a way to store files on the local network but also wanted it to be an active backup of all of his desktop files.
While there are more stream line ways to allow network access to files, the remote backup aspect meant that a NAS system would be most benefical. So I installed for them a small 4 TB NAS unit that connected
via Ethernet to the company's main computer (This is a small work-from-home company which meant there was little need for a larger or more compelex system). Spent some time troubleshooting the active-backup
feature and setting up a login system so that other people on the network could access the files, but after updating some drivers on the older computers and making sure the correct permissions were in place
it worked smoothly. The only edit that has been made to this system has been to use a 3rd party software to have a system-wide backup scheduled for a specific day in the month.

- Permission editing
